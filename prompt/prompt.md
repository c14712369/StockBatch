# 台股開盤前掃描｜完整 Prompt（免調整版）

> 使用方式：每個交易日早上 08:00 直接貼入，無需修改任何內容。

---

## 第一步：環境偵測（分析前必須先執行）

請依序完成以下四項偵測，結果將作為後續所有任務的分析基礎：

**① 日期確認**
自動抓取今日日期，確認為台灣時間工作日（週一至週五）。
若今日為假日或休市日，請直接告知「今日休市」，停止執行後續任務。

**② 主流題材偵測**
搜尋「今日台股強勢族群」、「本週投信重點買超族群」、「台股主流題材」。
從搜尋結果中，自動選出當下市場最強的 1～2 個核心題材，作為本次任務二的分析範圍。
輸出格式：
```
當日主流題材：[族群A] / [族群B]
判斷依據：（一句話說明，例：投信連三日買超集中AI伺服器族群）
```

**③ 大盤基調判斷**
搜尋昨日加權指數收盤漲跌幅，以及今晨台指期貨夜盤方向。
根據以下規則自動標註基調：
- 加權昨漲 > 0.5% 且期貨夜盤正價差 → 多頭攻擊
- 加權昨跌 > 0.5% 且期貨夜盤負價差 → 偏空防守
- 其餘情況 → 震盪整理

輸出格式：
```
大盤基調：[多頭攻擊 / 震盪整理 / 偏空防守]
加權昨日：+X.XX% ／ 期貨夜盤：正/負價差 X 點
```

**④ 國際指標自動對應**
根據②偵測到的主流題材，自動匹配以下對應指標並搜尋昨日收盤數據：

| 主流題材 | 對應指標 |
|---------|---------|
| AI伺服器 / 散熱 / 光通訊 / CPO | 輝達 NVDA 昨日漲跌幅% |
| 記憶體 / HBM / DRAM | 美光 MU 昨日漲跌幅% |
| 半導體設備 / 矽光子 / 先進封裝 | 費城半導體指數 SOX 昨日漲跌幅% |
| 非科技族群（生技/電池/傳產） | VIX 恐慌指數數值 |
| 題材橫跨多類 | 以上全部搜尋，取相關者 |

自動觸發規則（帶入任務四排名計算）：
- 對應指標單日跌幅 > 2% → 該族群標的在任務四自動降一個排名
- 對應指標單日漲幅 > 2% → 該族群標的在任務四自動升一個排名
- VIX > 25 → 所有標的籌碼權重下調至 20%，型態權重上調至 40%

輸出格式：
```
對應指標：[指標名稱] 昨日 [+/-X.XX%] → 任務四 [升一名 / 降一名 / 不調整]
```

## 第二步：數據獲取（環境偵測完成後執行）

請搜尋以下三類數據，每項標註資料來源與資料時間：

**① 三大法人盤後數據**
- 投信買超前 20 名個股（含買超張數）
- 外資買超前 20 名個股（含買超張數）
- 投信連續買超 3 日以上的個股清單

**② 個股技術數據**
針對①出現的個股，搜尋其：
- 最新一個交易日（昨日）收盤價、成交量、5 日均量
- 是否站上 5MA / 10MA / 20MA / 60MA / 126MA

**③ 族群強度數據**
- 最新一個交易日台股各類股漲跌幅排行
- 第一步②偵測到的主流題材族群，列出成分股漲跌幅前五名

⚠️ **數據誠信規則（強制執行）**
- 每項數據必須標註來源網站與資料時間
- 若某項搜尋結果為空、模糊或超過 2 個交易日前，明確標註「數據缺失」
- 禁止以歷史數據代替今日數據
- 禁止因表格欄位需要而填入未經搜尋確認的數字
- 數據缺失時，該欄位填入「—」，不得留白或填推估值

## 第三步：分析任務

### 角色設定

你是具備 20 年資歷的台股量化短線交易專家，專長為動能交易與籌碼追蹤。
輸出只含數據與邏輯論證，禁止使用「可能」「或許」「預計」「應該」等模糊詞彙。
每個結論必須對應至少一項數據支撐，不得只憑題材面做判斷。

---

### 任務一｜強勢鎖碼掃描

從第二步數據中，篩選近 3～5 個交易日同時符合以下三條件的個股：

**籌碼條件**
- 投信連續買超 ≥ 3 日
- 單日買超張數 ÷ 日均量 ≥ 10%

**技術條件**
- 股價突破近 20 日箱型整理上緣
- 收盤同時站上 5MA / 10MA / 20MA / 60MA（四線全過）

**強度條件**
- 近 3 日個股漲幅 > 同期加權指數漲幅（逆勢抗跌或順勢噴發皆符合）

**輸出格式：**

| 代號 | 名稱 | 投信連買天數 | 買超佔日均量% | 突破壓力位 | 投信成本區 | 防守支撐位 | 強度評級 | 數據來源 |
|-----|-----|------------|-------------|----------|----------|----------|--------|--------|
| ... | ... | ... 日 | ...% | ... 元 | ... 元 | ... 元 | ★★★☆☆ | ... |

⚠️ 若符合條件個股少於 3 支，直接說明「今日強勢鎖碼標的不足 3 支」，禁止降低篩選標準湊數。

---

### 任務二｜潛力標的掃描

分析範圍：**全市場**，不限制題材或族群。主流題材僅作市場背景參考，非篩選必要條件。

從第二步數據出發，找出具備上漲動能的標的：

**1. 強勢領頭股（列 1～2 支）**
不限族群，找出近期漲幅最強、成交量同步放大、且站上所有均線的個股。
可以是主流題材成分股，也可以是逆勢強攻的冷門股。

**2. 補漲潛力股（列 2～3 支）**
符合以下條件的標的（不限族群）：
- 近一季營收年增率為正
- 股價尚未突破近 3 個月高點（仍有上漲空間）
- 近 3 日投信有買超紀錄

**3. 即時催化劑**
搜尋以上個股近 14 日內是否有：法說會、國際展覽、大廠認證、重大訂單公告、轉投資利多。
若無，標註「近期無明確催化劑」。

**結論（必填）：**
直接點名 1 支最具轉機潛力的標的，說明動能來源與理由，限 50 字內。

---

### 任務三｜當日攻擊盤掃描

根據第二步數據，找出今日同時符合以下三條件的個股：

**量能條件**
今日成交量 > 5 日均量 × 2

**型態條件**
- 收長紅 K 棒
- 收盤價 ≥ 當日最高價 × 97%
- 收盤價突破 126MA（半年線）壓力

**籌碼條件**
外資與投信同步由賣轉買（雙向翻多首日，即昨日至少一方為賣超，今日兩者皆為買超）

**輸出格式：**

```
【代號｜名稱】
- 量能：今日 X 張 vs 5日均量 X 張（X 倍）
- 型態：收 +X.X%，收盤 X 元，突破 126MA 壓力位 X 元
- 籌碼：外資 +X 張（昨 -X 張），投信 +X 張（昨 -X 張）
- 爆漲理由：（一句話，限 30 字，必須含數據）
```

⚠️ 若今日無符合條件個股，直接說明「今日無攻擊盤標的」，禁止降低條件。

---

### 任務四｜明日開盤前五強標的

整合任務一至三的結果，加入第一步③④的大盤基調與指標調整，篩選明日最具攻擊動能的 5 支標的。

**篩選權重（基礎值）**
- 有明確動能來源（主流題材 / 催化劑 / 法人連續佈局 / 逆勢強攻，擇一即符合）：40%
- 近 3 日投信連買且法人買超佔比放大：35%
- 股價位於所有均線上方且突破箱型壓力：25%

**自動調整規則**
- 若大盤基調為「偏空防守」：籌碼權重降至 25%，型態權重升至 35%，題材權重不變
- 若對應國際指標跌幅 > 2%：相關族群標的降一名
- 若對應國際指標漲幅 > 2%：相關族群標的升一名
- 若 VIX > 25：所有標的籌碼權重統一降至 20%，型態權重升至 40%

**輸出格式：**

⚠️ 若任務一至三合計符合條件標的不足 5 支，依實際數量輸出，禁止補充不符條件的標的。

| 排名 | 代號 | 名稱 | 動能來源 | 籌碼強度 | 型態分數 | 綜合評分 | 進場參考價 | 防守支撐位 | 排名調整說明 |
|-----|-----|-----|---------|---------|---------|---------|----------|----------|-----------|
| 1 | ... | ... | ... | ★★★★★ | ★★★★☆ | 92 | X 元 | X 元 | 無調整 |
| 2 | ... | ... | ... | ★★★★☆ | ★★★★★ | 89 | X 元 | X 元 | 因SOX跌2.3%降一名 |

## 附錄：使用說明

**每日執行時間**
建議 08:00 貼入，08:45 前完成核對，留 15 分鐘緩衝。

**建議配合的人工核對步驟（5 分鐘內完成）**
1. 確認任務四第 1、2 名的 K 線圖型態與均線位置
2. 核對投信成本區與防守支撐位是否與 K 線吻合
3. 確認當日開盤氣氛（台積電期貨、輝達開盤方向）

**此 Prompt 的邊界**
- ✅ 適合：技術面 + 籌碼面的短線動能操作
- ✅ 適合：開盤後前 30 分鐘的強勢追價判斷
- ❌ 不適合：波段操作或存股邏輯
- ❌ 不適合：基本面深度研究
